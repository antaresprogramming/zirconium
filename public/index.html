Zirconium

An opinionated CSS library of components. Offical CSS written by [Francis Rubio](https://github.com/maniczirconium/maniczirconium) for [Antares Programming](https://antaresprogramming.github.io/)

Goals

  • Standardize components for the website
  • Support dark mode from the get-go
  • Promote accessibility by using HTML structure as basis for styling in some components
  • Include minimal JavaScript (this is for a blog, not a website)

Elements

Buttons

This is the standard button. Styles are applied to button or a elements by giving them the .button class.

Button-styled Link 
<button class="button">Default Button</button>
<a href="#!" class="button">Button-styled Link</a>

Buttons can appear elevated by using shadows. Add shadows to buttons by using the shadow class.

Shadow button 
<button class="shadow button">Shadow Button</button>
<a href="#!" class="shadow button">Shadow button</a>

Icon buttons can be created by using either an <svg> or <img element. Left-side icons are placed as the first child of the button, while the right-side ones are placed as the last in the markup. Icons are also resized to 16px×16px images. 

<button class="button">
  <svg>…</svg>
  Left Icon Button
</button>
<button class="button">
  Right Icon Button
  <svg>…</svg>
</button>
<button class="button">
  <svg>…</svg>
</button>

Buttons can also appear flat by removing their background color on the default state. Do this by adding the .no-bg class. Note that .no-bg buttons have transparent backgrounds only on the default state. They will still be styled with background colors on hover, click, and focus. 

<button class="button accent">Default Button</button>

Buttons can be smaller by adding the .small class. This doesn't change the font size, but rather only the padding. 

<button class="button small">Small Button</button>
<button class="button small">
    <svg>…</svg>
    Small Button
</button>

.button.small buttons automatically revert to the normal button size in mobile devices so they get easily tapped by people with big thumbs. This behavior should not be overrided.

Buttons can also be larger by adding the .large class. This doesn't change the font size, but rather only the padding. 

<button class="large button">Large Button</button>

Accent buttons, used sparingly, are created by using the .accent class. 

<button class="accent button">Accent</button>

Alerts

Alerts are banners that present information to the user when something happens. Use cases include, but are not limited to, the following:

  • Accomplishment of a background task
  • Introduction of new features that need onboarding
  • Errors in the app
  • Warnings before the user decides on any vital action
  • Communicating the result of a user action

This is the default alert.

Note Make sure that alerts have role="alert" on them so assistive technologies will recognize them as actual alerts.

Lorem ipsum dolor sit amet.

<div class="alert" role="alert">
    <div class="alert__body">
    <p>Lorem ipsum dolor sit amet.</p>
    </div>
</div>

Alerts can also have titles, or headers. This example adds an h3 element to the alert as its heading.

Alert!

Lorem ipsum dolor sit amet.

<div class="alert" role="alert">
  <div class="alert__body">
    <h3>Alert!</h3>
    <p>Lorem ipsum dolor sit amet.</p>
  </div>
</div>

Closable alerts can add their own close buttons as the last child of the .alert element. Notice that pressing the button inside the .alert component does not automatically close the alert. You will have to handle this on your own script.

Default

This is a default alert

<div class="alert" role="alert">
  <div class="alert__body">
    <h3>Default</h3>
    <p>This is a default alert</p>
  </div>
  <button class="button small no-bg">
    <svg>&helip;</svg>
  </button>
</div>

Note: Close buttons are handled automatically by zirconium.js. Closable alerts can work without custom script by adding the data-close attribute to your close button. Just make sure that you include zirconium.js in your markup, and that your close button is an immediate child of .alert.

Default

This is a default alert

<div class="alert" role="alert">
  <div class="alert__body">
    <h3>Default</h3>
    <p>This is a default alert</p>
  </div>
  <button class="button small no-bg" data-close>
    <svg>…</svg>
  </button>
</div>
